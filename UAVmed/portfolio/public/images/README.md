# Dashboard Images

Place your dashboard screenshots here:

- `navigator-dashboard.png` - Navigator Dashboard screenshot
- `pilot-dashboard.png` - Pilot Dashboard screenshot  
- `ems-dashboard.png` - EMS Dashboard screenshot

## How to Add Images

1. Take screenshots of your dashboards from the main project
2. Save them as PNG files with the names above
3. Place them in this `public/images/` folder
4. The portfolio will automatically display them on slides 5, 6, and 7

## Image Recommendations

- Format: PNG (for transparency) or JPG
- Resolution: 1920x1080 or higher
- File size: Optimize for web (under 500KB each)
